Identification of candidate mobile mRNAs from rootstock to scion

To identify putative mobile mRNAs transported from the Lagenaria rootstock into the watermelon scion, trimmed RNA-seq reads from all samples were first aligned to the watermelon reference genome to remove transcripts originating from the scion. Reads that failed to align were considered non-watermelon sequences and were extracted as candidate non-scion reads. These unmapped reads were then aligned to the Lagenaria genome using BLASTN, and reads showing high-confidence matches were retained as potential rootstock-derived transcripts. Genomic coordinates of these aligned reads were intersected with Lagenaria gene annotations to assign corresponding gene IDs. To eliminate background signals and false positives, the same analysis was performed on homograft (WM/WM) control datasets, and any Lagenaria genes detected in these controls were removed from the heterograft results. Finally, only genes detected in at least two biological replicates of heterograft samples and absent from all homograft samples were retained as high-confidence candidates. Using this stringent filtering strategy, candidate mobile mRNAs originating from the rootstock and present in the scion were identified for both control and whitefly-infested conditions.

To further prioritize biologically relevant candidates, read support for each identified Lagenaria gene was quantified by counting the number of sequencing reads assigned to each gene across biological replicates. Read counts from replicate samples were summed within each condition and used to rank candidate mobile mRNAs according to total read support, allowing identification of highly abundant and reproducibly detected transcripts. Condition-specific mobile mRNAs were determined by comparing gene lists between control and whitefly-infested treatments, enabling classification of shared, control-specific, and stress-specific mobile transcripts. The highest-ranked genes based on read support were selected as strong candidate mobile mRNAs for downstream functional analysis and experimental validation.

In the updated workflow, I first constructed a gene-by-sample read count matrix using all homograft and heterograft samples. These counts were then normalized using DESeq2 size-factor normalization to account for differences in library size and sequencing depth across samples.

Using the normalized counts, I calculated the average expression levels for homograft and heterograft groups separately for both control and whitefly conditions. Fold change was then computed as the ratio of heterograft abundance to homograft abundance.

Genes showing at least a 5-fold enrichment in heterografts relative to homografts were retained as candidate mobile genes. To further improve confidence, I required that each gene be detected in at least two heterograft replicates and removed genes present in the homograft false-positive sets.

The attached script implements these steps: normalization, fold-change calculation, replicate support filtering, and false-positive removal. The resulting gene lists represent high-confidence candidate mobile mRNAs under control and whitefly infestation conditions.

Summary of Analysis Performed Based on Dr. Lee’s Suggestions
To improve the identification of mobile RNAs, an additional filtering strategy was implemented following Dr. Lee’s recommendations. The goal was to avoid losing biologically meaningful mobile RNAs that may show low background expression in homograft samples but are strongly enriched in heterografts.

1. Generation of Gene-Level Read Counts for All Samples
Gene-level read support was first quantified for each RNA-seq sample by counting the number of sequencing reads mapped to each Lagenaria gene.
Previously, gene read counts had already been generated for the heterograft samples. To enable comparison between graft combinations, gene read counts were additionally generated for the corresponding homograft samples.

As a result, gene count tables were available for all experimental samples:
Condition	Homograft Samples	Heterograft Samples
Control	S4, S5, S6	S10, S11, S12
Whitefly infestation	S19, S20, S21	S25, S26, S27

Each table contained the number of reads supporting each gene in that sample.

2. Summation of Gene Counts Across Biological Replicates
To obtain a robust measure of gene support under each graft condition, read counts from biological replicates were combined.

Replicate counts were summed separately for:
Control homograft samples
Control heterograft samples
Whitefly homograft samples
Whitefly heterograft samples

This produced four summarized gene count tables representing the total read support per gene under each graft condition.

3. Calculation of Fold Change Between Heterograft and Homograft Samples
To implement Dr. Lee’s recommendation, fold change enrichment was calculated for each gene by comparing its abundance between heterograft and homograft samples.

Fold change was calculated as:
Heterograft read count / Homograft read count

This comparison was performed separately for:
Control condition (heterograft vs control homograft)
Whitefly condition (heterograft vs whitefly homograft)

This step ensured that genes showing strong enrichment in heterografts were retained even if they exhibited low background expression in homografts.

4. Application of Fold-Change Enrichment Filter
Genes were retained only if they showed strong enrichment in heterograft samples.

Two thresholds were examined:
≥5-fold enrichment
≥10-fold enrichment

For downstream analysis, the ≥5-fold enrichment threshold was selected to retain genes showing substantial heterograft-specific signal.

5. Integration with Replicate-Supported Mobile RNA Candidates
Prior steps in the pipeline had already identified candidate mobile RNAs based on their presence in heterograft samples. To increase confidence in mobile RNA detection, only genes detected in at least two heterograft biological replicates were retained. This replicate filtering step reduces stochastic or low-confidence detections.

6. Removal of False Positives Detected in Homograft Controls
Genes detected in homograft samples were previously classified as potential false positives. These genes may arise from background mapping or endogenous gene expression rather than true RNA mobility.
To ensure high-confidence mobile RNA detection, genes identified as false positives were removed from the candidate lists.

7. Final Identification of High-Confidence Mobile RNAs
The final set of mobile RNA candidates was defined by integrating all filtering criteria:

A gene was considered a mobile RNA only if it satisfied all of the following:
Detected in at least two heterograft replicates
Not present in the homograft false-positive set
Enriched in heterograft samples relative to homografts (≥5-fold)
